# Tic-Tac-Toe Game

A simple console-based Tic-Tac-Toe game built using Python.

## Features
- Human vs Computer
- Random computer moves
- Win detection
- Tie detection
- Input validation

## Technologies Used
- Python 3

## How to Run

```bash
python main.py
```
